EXPLICACIÓN DEL PROBLEMA
================================================================================

Los 6 artículos del blog listados arriba tienen un problema técnico donde:

✅ SÍ se muestra:
   - Título del artículo
   - Fecha de publicación
   - Resumen/extracto
   - Tags
   - Formulario de contacto

❌ NO se muestra:
   - Contenido completo del artículo (texto, párrafos, información detallada)
   - El cuerpo del artículo está vacío

CAUSA TÉCNICA:
El sitio usa React con carga dinámica. Los datos del artículo no se están
pasando correctamente al componente que renderiza el contenido. Probablemente
falta la propiedad "content" o "body" en los datos, o el componente no
está configurado para mostrarla.

================================================================================
SOLUCIÓN RÁPIDA
================================================================================

1. Revisar el archivo de datos del blog (blogData-BY5NjuyS.js)
2. Verificar que cada artículo tenga la propiedad "content" con el HTML
3. Revisar el componente BlogPost para que renderice el contenido:

   <div dangerouslySetInnerHTML={{ __html: post.content }} />
