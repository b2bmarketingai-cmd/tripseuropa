## V. ERRORES CRÍTICOS A EVITAR

### Error 1: Traducción Automática sin Revisión

❌ **PROBLEMA**: Google Translate genera:
- "Haga clic" → "Clique aqui" (pero está en potencial traducir mal contexto)
- "Éxito" → "êxito" (incorrecto, debe ser "sucesso")
- "Reserva" → "Reservação" (ambos válidos pero inconsistente)

✅ **SOLUCIÓN**: 
- Revisar con nativo portugués
- Crear glossario terminológico (anexo)
- Usar memoria de traducción (TMX)

***

### Error 2: Inconsistencia Terminológica

❌ **PROBLEMA**: Usar diferentes palabras para lo mismo:
- Página 1: "Pacote turístico"
- Página 2: "Pacote de viagem"
- Página 3: "Pacote de férias"

✅ **SOLUCIÓN**:
```
GLOSSARIO CENTRALIZADO:
Concepto → Traducción oficial
─────────────────────────
Paquete → Pacote de viagem (UNIFORME)
Destino → Destino (UNIFORME)
Reserva → Reserva (UNIFORME)
```

✅ **CORRECTO**:
```html
<title>Acomodação em São Paulo - Trips Europa</title>
```

***

### Error 4: Confusión Brasil vs Portugal

❌ **PROBLEMA**: Usar "autocarro" (Portugal) en web de Brasil:
```html
<p>Reserve seu autocarro para o passeio</p>
```

✅ **CORRECTO**: Usar "ônibus" (Brasil):
```html
<p>Reserve seu ônibus para o passeio</p>
```
